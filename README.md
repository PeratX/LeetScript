LeetScript
===========

__运行于 [SimpleFramework](https://github.com/iTXTech/SimpleFramework) 和 [LeetQQ](https://github.com/PhQAgent/LeetQQ)，用于执行 LeetScript 的解释器。__

介绍
-----------
通过 __LeetScript__ ，开发者能够编写简单的，基于菜单操作的QQ机器人
